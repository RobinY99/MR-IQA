# MR-IQA

**MR-IQA: A Unified Margin View for Image Quality Assessment**

This repository will host the official code and resources for **MR-IQA**, a unified margin-based framework for image quality assessment.

## Overview

MR-IQA studies image quality assessment from a unified margin perspective. The project is designed to support reproducible training, validation, and generalization evaluation across multiple IQA datasets.

More details will be added after the paper and code are finalized.
